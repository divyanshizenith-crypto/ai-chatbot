**AI Chatbot**
This project is an AI chatbot built with Python after completing Harvard's CS50P (Introduction to Programming with Python) course. It demonstrates the programming concepts, problem-solving techniques, and software development skills learned throughout the course.
The chatbot can interact with users through natural language conversations and showcases the practical application of Python fundamentals, including functions, loops, conditionals, data structures, file handling, and modular programming.

**Features**
> Interactive conversational interface
> Natural language response generation
> Clean and modular Python code
> Beginner-friendly project structure
> Built as a hands-on application of CS50P concepts
**About**
This project was created to apply and strengthen the Python programming skills gained from Harvard's CS50P course while exploring the fundamentals of AI-powered chatbots.
